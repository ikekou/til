# make gitigonore with gibo from gitignore.io

アップデート
```bash
gibo update
```

リストアップ
```bash
gibo list
```

macでinttellij ideaを使ったウェブプロジェクトの.gitignoreを作る
```bash
gibo dump JetBrains macOS Node >> .gitignore
```
